# Front-End Template

## Development

Install dependencies:

```
$ yarn
```

Build:

```
$ yarn build
```

Start Browsersync server and file watching:

```
$ yarn start
```

Run `yarn build` and `yarn start` simultaneously:

```
$ yarn development
```

or

```
$ yarn dev
```

## Production

Install dependencies:

```
$ yarn
```

Build:

```
$ yarn build:production
```

or

```
$ yarn prod
```
